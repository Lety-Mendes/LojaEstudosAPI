# QA Automation: Validação Completa de API REST Lojinha

## Objetivo
Este projeto visa demonstrar habilidades em Automação de Testes de API para garantir a qualidade dos principais módulos da API REST Lojinha.

## Tecnologias Utilizadas
* Java
* REST Assured
* JUnit 5
* Maven

## Documentação
A especificação da API (OpenAPI/Swagger) utilizada como contrato para estes testes está incluída neste repositório em `docs/swagger.yml`.

## Como Executar

Para rodar os testes localmente, certifique-se de ter instalado:

* **JDK (Java Development Kit)**: Versão 17 ou superior.
* **Apache Maven**: Para gerenciamento de dependências e execução dos testes.
